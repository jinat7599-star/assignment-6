💠 DigiTools Platform

A modern style Web application. Where users can explore and purchase digital tools with a smooth cart experience.

🔧 Technologies Used

* React.js
* Tailwind CSS
* JavaScript (ES6+)
* React Toastify

✨ Features

*  Add to Cart Option with live update
*  Toggle between Products & Cart
*  Toast notifications for actions
*   Responsive design
